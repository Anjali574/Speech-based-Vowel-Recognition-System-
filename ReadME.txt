Sample And Test Data
1. 100 recordings are provided for the project(20 for each vowel). Sample data in the folder 'AS4_190101012\Training' and Test data in the folder 'AS4_190101012\AS4_190101012\Test'.
2. The reference files are created and stored into in the folder 'AS4_190101012\Vowel_Reference' (1 references for each vowel).

How to Run:-
1. Open project 'AS4_190101012' in Visual Studio 2010.
2. Open file 'AS4_190101012.cpp'. 
3. Build project using 'Build Solution' under 'Build' option.
4. Run project using 'Start Without Debugging' under 'Debug' option.
5. Terminal will open, and will take some time for 'Training and Installation'.
6. After that the outputs of vowel recognized corresponding to the different test data in the terminal is shown.
